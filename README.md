# AI/ML Portfolio

Minimal portfolio website for AI/ML engineers.

## Folder Structure

```
portfolio/
├── index.html          # Main HTML
├── css/
│   └── style.css       # All styles + design tokens
├── js/
│   └── main.js         # Canvas animation, scroll reveal
├── assets/
│   └── resume.pdf      # Add your resume here
└── README.md
```

## Deploy to GitHub Pages (Free Hosting)

1. Create a new GitHub repo named: `yourusername.github.io`
2. Upload all files to the repo
3. Go to **Settings → Pages → Source: main branch**
4. Your live URL: `https://yourusername.github.io`
